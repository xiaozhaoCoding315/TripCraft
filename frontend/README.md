# TripCraft Frontend

React + Vite frontend for the TripCraft travel planning system.

## Quick start

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

The dev server proxies `/api` and WebSocket traffic to `http://localhost:8000`.

Set `VITE_AMAP_WEB_JS_KEY` to enable real Amap JS rendering in future map iterations.
